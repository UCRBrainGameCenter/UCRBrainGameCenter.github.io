Detailed Description

A class to manage a set of parameterized pools. Ex: A pool of float[] buffers pooled by length.

Template Parameters
TThe underlying pooled object type. Ex: float[] for simple buffer.
TArgThe type on which the pools are parameterized. Ex: int for array length.
