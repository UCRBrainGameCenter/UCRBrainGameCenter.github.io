Detailed Description

Implementation of bag data structure. Refer to https://algs4.cs.princeton.edu/13stacks/ for notes on how it suspposed to function and how this class can be improved

Remarks
{This class is okay but not perfect. My particular issue with it is that I cannot clone the templates given. Cloning the templates would be ideal because we can keep the clean list safe from outside damage through modification. Inherently the code is less safe with this data structure if the programmer is not paying attention and making sure to program in cloning after a Pull and also not modifying the passed in list.}
